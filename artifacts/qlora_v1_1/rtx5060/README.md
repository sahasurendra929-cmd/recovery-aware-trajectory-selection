# RTX 5060 QLoRA v1.1 three-arm results

This directory contains the compact, auditable QLoRA **v1.1** results. It is not a v2 result set and does not modify or supersede any v2 artifact.

## Scope and claim boundary

- Hardware: NVIDIA GeForce RTX 5060 Laptop GPU, 8 GB class (8,546,484,224 bytes; 8,151 MiB reported by NVIDIA tooling).
- Driver: 591.86.
- Source repository commit: `b7cc97ae6b9ca0aca9530c25f34b5dc20f6f0563`.
- Frozen tau-bench commit: `59a200c6d575d595120f1cb70fea53cef0632f6b`.
- All three arms completed training and a full 900-example held-out evaluation.
- These results measure offline held-out next-tool-call/tool-action SFT performance only. They do not demonstrate end-to-end Agent success.
- v1.1 results must not be directly mixed with results produced using a different evaluator protocol.

The copied metrics, run manifests, selection manifests, YAML configuration, and environment JSON are byte-for-byte copies of the local run artifacts. No metric was edited or recomputed for this upload.

## Files not generated by the v1.1 runner

The local runs did not produce `predictions.jsonl`, `training_metrics.json`, `training_log.json`, `command.txt`, `console.log`, or `evaluation.log`. They are not fabricated in this handoff. Training completion is evidenced by the final checkpoint retained locally and the completed `run_manifest.json`; evaluation completion is recorded by each validated 900-example `metrics.json`.

`UPLOAD_MANIFEST.json` inventories every other uploaded file. It does not list itself because embedding its own final SHA-256 would be self-referential.
